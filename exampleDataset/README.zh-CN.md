# 怎么构造数据集
1. 首先必须确保数据集文件格式为 csv 格式
2. 必须有一列是时间戳列
3. 其他列必须是数值类型（int,float等,因为其他类型暂时没有时间完成）

------------------------------------------------  

像下面这个表，年龄作为时间戳，金钱是一个可能随着年龄这个时间戳变化的变量
|age|money|
|:---:|:---:|
|10|20000000|
|15|30000000|
|20|28000000|
|30|20000700|
|35|40000000|
|50|9000000|
